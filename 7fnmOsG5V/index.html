[Go] 5.函数

2021-02-02 Golang

定义

函数是语句序列的集合，能够将一个大的工作分解为小的任务，对外隐藏了实现细节。
函数组成：

  • 函数名
  • 参数列表(parameter-list)
  • 返回值(result-list)
  • 函数体(body)
func name(参数列表)(返回值列表){
    函数体
}

单返回值函数：

func plus(a,b int)(res int){
    return a+b
}

多返回值函数：

func multi()(string, int){
    return "name", 18
}

命名返回值：

// 被命名多返回参数的值为该类型的默认零值。
// 该例子中 name 默认初始化为空字符串， height 默认初始化为 0
func namedReturnValue()(name string, height int){
    name = "xiaoming"
    height = 180
    return
}

**注：**说白了就是给出了返回值的参数名，例如例子中给出了"name"和"height"，如果在函数中需要用到这两个参数，则它们的初始值是它们类型对应的默认零值，例如这里name就是""(空)，height就是0。
例如下面这个例子，设置了返回值的参数名后，在函数中给name赋值为"test" ，而height没有赋值，所以返回的时候依然是int的零值，即0。所以打印后输出为"test 0"

参数可变函数：

func sum(nums ...int)int{   // 用三个点，代表这个参数的数量是不固定的。
    fmt.Println("len of nums is：",len(nums))  // 这里的len(nums)就是传递过来的参数格式，所以这里nums其实是个列表
    res :=0
    for _, v := range nums{
        res += v
    }
    return res
}

func main(){
    fmt.Println(sum(1))
    fmt.Println(sum(1,2))
    fmt.Println(sum(1,2,3))  // 传递几个参数，函数就接收几个
}

执行结果如下：

len of nums is： 1
1
len of nums is： 2
3
len of nums is： 3
6

匿名函数：

  • Go 支持匿名函数，匿名函数就是没有名字的函数
  • 如果我们某个函数只是希望使用一次，可以考虑使用匿名函数
  • 匿名函数也可以实现多次调用。
  • 全局匿名函数，将匿名函数赋值给一个全局变量，再通过该全局变量进行调用。
    匿名函数使用方式 1
    在定义匿名函数时就直接调用，这种方式匿名函数只能调用一次
res1 := func (n1 int, n2 int) int {
    return n1 + n2
}(10, 20)  // 定义完直接调用

匿名函数使用方式 2
将匿名函数赋给一个变量(函数变量)，再通过该变量来调用匿名函数
注：这个和直接用func定义没啥区别，如果硬要说区别，那就是性能区别，但匿名函数如果比较简单会被编译器优化成inline函数，那样就没啥区别了。

a := func (n1 int, n2 int) int {
    return n1 - n2
}     
res2 := a(10, 30)
res3 := a(90, 30)

匿名函数使用方式 3
如果将匿名函数赋给一个全局变量，那么这个匿名函数，就成为一个全局匿名函数，可以在整个程序有效。

var (
    //Fun1就是一个全局匿名函数
    Fun1 = func (n1 int, n2 int) int {   //这里Fun1的定义不需要“:”，因为只有在函数体内部才需要冒号。
        return n1 * n2
    }
)

func main() {
    //全局匿名函数的使用
    res4 := Fun1(4, 9)
    fmt.Println("res4=", res4)
}

闭包
简单理解：

  • 闭包就是能够读取其他函数内部变量的函数。
  • 只有函数内部的子函数才能读取局部变量，因此可以把闭包简单理解成”定义在一个函数内部的函数”。```
package main

import "fmt"

func getSequence() func() int {
   i:=0
   return func() int {
      i+=1
     return i  
   }
}

func main(){
   /* nextNumber 为一个函数，函数 i 为 0 */
   nextNumber := getSequence()  

   /* 调用 nextNumber 函数，i 变量自增 1 并返回 */
   fmt.Println(nextNumber())
   fmt.Println(nextNumber())
   fmt.Println(nextNumber())
   
   /* 创建新的函数 nextNumber1，并查看结果 */
   nextNumber1 := getSequence()  
   fmt.Println(nextNumber1())
   fmt.Println(nextNumber1())
}

函数作为参数

func sayHello(name string){
    fmt.Println("Hello", name)
}

func logger(f func(string), name string){  // 这里的f func(string)的意思是参数f,它的类型是一个接收string类型参数的函数。
    fmt.Println("start calling method sayHello")
    f(name)
    fmt.Println("end calling method sayHellog")
}

func main(){
    logger(sayHello, "鸣人")  //这里将sayHello函数作为参数值，传递给f，将字符串“鸣人”传递给name
}

传值和传引用
注：指针中&是取地址，*是取值

func sendValue(name string){
    name = "naruto"
}

func sendAddress(name *string){
    *name = "naruto"
}

func main(){
    // 传值和传引用
    str := "鸣人"
    fmt.Println("before calling sendValue, str：",str)
    sendValue(str)
    fmt.Println("after calling sendValue, str：",str)

    fmt.Println("before calling sendAddress, str：",str)
    sendAddress(&str)  // 这里将str的地址传给了值指针name，相当于让*name指向&str上的值，所以修改了*name的值，就是修改了str的值
    fmt.Println("after calling sendAddress,str：",str)
}

执行结果：

before calling sendValue, str： 鸣人
after calling sendValue, str： 鸣人
before calling sendAddress, str： 鸣人
after calling sendAddress,str： naruto
