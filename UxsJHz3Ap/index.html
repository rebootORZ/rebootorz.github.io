[Go] 1.切片

2021-02-02 Golang

切片组成要素：

指针：指向底层数组
长度：切片中元素的长度，不能大于容量
容量：指针所指向的底层数组的总容量

常见初始化方式：

1.使用make初始化
slice := make([]int,5) //初始化长度和容量都为5的切片
slice := make([]int,5,10) //初始化长度为5，容量为10的切片

2.使用简短定义
slice := []int{1,2,3,4,5} //[]中没有指定，这样的是切片，指定的话是数组。长度、容量都是5，具体看下图

3.使用数组来初始化切片
arr := [5]int{1,2,3,4,5}
slice := arr[0:3] //左闭右开区间，最终为[1 2 3] 注意，这里是中括号，元素之间是空格

4.使用切片来初始化切片
sliceA := []int{1,2,3,4,5}
sliceB := sliceA[0:3] //左闭右开区间，最终sliceB为[1 2 3]

长度和容量

package main

import (
    "fmt"
)

func main(){
    slice := []int{1,2,3,4,5}
    fmt.Println("len: ",len(slice))
    fmt.Println("cap: ",cap(slice))

    //改变长度
    slice = append(slice,6) //长度变为6，而不是增加6
    fmt.Println("after append operation: ")
    fmt.Println("len: ",len(slice))
    fmt.Println("cap: ",cap(slice)) //注意，当改变长度后，导致来底层数组容量不足时，会自动重新非配数组的空间，通常是当前切片长度的两倍。
}

注意点

如果多个切片共享一个底层数组，当一个切片的改变，导致底层数组的改变，或者是直接对底层数组的修改，将会影响上层多个切片的值。

package main

import(
    "fmt"
)

func main(){
    slice := []int{1,2,3,4,5}
    newSlice := slice[0:3]
    fmt.Println("befor modifying underlying array:") //修改底层数组之前
    fmt.Println("slice: ",slice)
    fmt.Println("newSlice: ",newSlice)
    fmt.Println()

    newSlice[0] = 6
    fmt.Println("after modifying underlying array:")  //修改底层数组之后
    fmt.Println("slice: ",slice)
    fmt.Println("newSlice: ",newSlice)
}

执行结果：

befor modifying underlying array:
slice:  [1 2 3 4 5]
newSlice:  [1 2 3]

after modifying underlying array:
slice:  [6 2 3 4 5]
newSlice:  [6 2 3]

使用copy方法可以避免共享同一个底层数组

package main

import(
    "fmt"
)

func main(){
    slice := []int{1,2,3,4,5}
    newSlice := make([]int,len(slice)) //上一个代码中用的是“切片生成切片”，它们都是指向同一个底层数组。而这里是用make重新生成了一个切片，有自己指向的底层数组
//copy函数：
//1.不同类型的切片无法复制
//2.如果s1的长度大于s2的长度，将s2中对应位置上的值替换s1中对应位置的值
//3.如果s1的长度小于s2的长度，多余的将不做替换   
    copy(newSlice,slice)  //使用slice的值替换newSlice的值
    fmt.Println("befor modifying underlying array:") //修改底层数组之前
    fmt.Println("slice: ",slice)
    fmt.Println("newSlice: ",newSlice)
    fmt.Println()

    newSlice[0] = 6
    fmt.Println("after modifying underlying array:")  //修改底层数组之后
    fmt.Println("slice: ",slice)
    fmt.Println("newSlice: ",newSlice)
}

输出结果如下：

befor modifying underlying array:
slice:  [1 2 3 4 5]
newSlice:  [1 2 3 4 5]

after modifying underlying array:
slice:  [1 2 3 4 5]
newSlice:  [6 2 3 4 5]
