[Go] 跨平台编译

2021-02-08 Golang

说明

跨平台编译操作很简单，具体原理还没去了解，但是这里有个坑，做下记录。

跨平台编译命令如下：

**注：**对应的位置填入需要编译的类型即可。

Mac 下编译 Linux 和 Windows 64位可执行程序

CGO_ENABLED=0 GOOS=windows GOARCH=amd64 go build  main.go

CGO_ENABLED=0 GOOS=linux GOARCH=amd64 go build main.go

Linux 下编译 Mac 和 Windows 64位可执行程序

CGO_ENABLED=0  GOOS=darwin GOARCH=amd64 go build main.go

CGO_ENABLED=0 GOOS=windows GOARCH=amd64 go build main.go

Windows 下编译 Mac 和 Linux 64位可执行程序

SET CGO_ENABLED=0
SET GOOS=darwin
SET GOARCH=amd64
go build main.go

SET CGO_ENABLED=0
SET GOOS=linux
SET GOARCH=amd64
go build main.go

PS:网上很多教程里面，MAC和LINUX下的编译命令都是分行写的，测试下来发现这样执行并不能成功，写成一行就成功了。ORZ
